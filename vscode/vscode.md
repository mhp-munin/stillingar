### Navigation ###

| Command          | Key Combo                  | Note   |
| ---------------- | -------------------------- | ------ |
| Toggle sidebar | `<space>` + `e` ||
| Show all editors | `<space>` + `,` ||

### Coding ###

| Command          | Key Combo                  | Note   |
| ---------------- | -------------------------- | ------ |
| Move lines down | `<shift>` `j` | In Visual mode |
| Move lines up | `<shift>` `k` | In Visual mode |
| Show hover window | `<shift>` `k` | Only in Normal mode |
